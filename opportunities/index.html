    Mentorships & Internships

    Google Summer of Code

    An international annual program in which Google awards stipends to contributors who successfully complete a free and open-source software coding project during the summer.

    LFX Mentorship

    Get involved in the Linux Foundation’s leading mentorship programme, designed to help train the next generation of open source developers, supported through sponsorship and specialised industry training.

    (Currently being adapted for participants in the Southern Hemisphere).

    Training & Development

    KCNA Essentials Course

    This course provides an overview of cloud native technologies and how container orchestration systems like Kubernetes can help to implement and maintain them.

    *Ask about sponsored access for Māori cohorts

    Kubernetes Cloud Native Associate (Exam)

    The KCNA exam demonstrates a user’s foundational knowledge and skills in Kubernetes and the wider cloud native ecosystem.

    *Ask about sponsored access for Māori cohorts

    EDX Essentials Course

    Want to learn Kubernetes for free? Get an in-depth primer on this powerful system for managing containerised applications.
